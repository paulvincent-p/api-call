api-call project (using Postman)

A simple project where I used Postman to practice working with APIs. I tried out the five basic HTTP methods like GET to fetch data, POST to add new data, PUT to fully update a record by ID, PATCH to partially change it, and DELETE to remove it.

Setup:
1. Download and install Postman
2. Set up the request endpoints
3. Send requests using each HTTP method and check the responses
