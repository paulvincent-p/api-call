## api-call
 
## Description
A simple project where I used Postman to practice working with APIs. I tried out the five basic HTTP methods such as GET to fetch data, POST to add new data, PUT to fully update a record by ID, PATCH to partially change it, and DELETE to remove it.

## Setup

**Requirements**
- Postman VS Code extension
- [Laravel Herd](https://herd.laravel.com/) 
- [Composer](https://getcomposer.org/)

**GitHub Fork/Clone**
- Fork the repository ([here is the guide](https://help.github.com/articles/fork-a-repo/))
- Clone to your machine
```git clone https://github.com/paulvincent-p/api-call.git```
- Or download the repository directly [here](https://github.com/paulvincent-p/api-call/archive/refs/heads/main.zip)

**Running the Project**
1. Open the project folder.
2. Run `composer install`.
3. Copy `.env.example` to `.env` and set your database details.
4. Run `php artisan key:generate`.
5. Run `php artisan migrate`.
6. Start the local server with `php artisan serve`.
